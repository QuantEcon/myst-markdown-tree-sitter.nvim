# MyST Markdown Configuration Examples

This directory contains example configurations for various use cases.

## Examples

- **[basic.lua](basic.lua)** - Minimal setup (recommended starting point)
- **[advanced.lua](advanced.lua)** - Full configuration with all options
- **[performance.lua](performance.lua)** - Optimized for large files
- **[jupyter.lua](jupyter.lua)** - Jupyter notebook integration
- **[lazy-nvim.lua](lazy-nvim.lua)** - Complete lazy.nvim plugin spec
- **[with-lsp.lua](with-lsp.lua)** - Integration with LSP and other tools

## Usage

Copy the relevant example to your Neovim configuration file (usually `~/.config/nvim/init.lua` or `~/.config/nvim/lua/plugins/myst-markdown.lua` for lazy.nvim).

Adjust settings as needed for your workflow.
